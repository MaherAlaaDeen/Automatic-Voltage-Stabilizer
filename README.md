# Automatic-Voltage-Stabilizer
This repository contains the documentation and design files for an Automatic Voltage Stabilizer developed as a project for CCE 301L at the American University of Science & Technology
## Overview
The project focuses on designing and implementing an Automatic Voltage Stabilizer using relay-based technology to regulate voltage fluctuations and ensure stable power supply to electronic devices.
## Key Features
* Relay-Based Technology: Utilizes electromechanical relays and an autotransformer for voltage regulation\
* Over/Under Voltage Protection: Protects connected devices from voltage fluctuations.
* LCD Display: Provides real-time input and output voltage monitoring.
* Modular Design: Consists of distinct power supply and stabilizer modules for easier maintenance and scalability.
* Automatic Cut-Off: Safeguards against extreme voltage conditions to prevent damage.

## Objectives

1- Understanding Voltage Stabilization: Explaining the principles and types of voltage stabilizers.

2- Design and Simulation: Creating a design and simulating the functionality of the stabilizer.

3- Implementation and Testing: Building the stabilizer prototype and conducting performance tests.

## Tools and Components Used
* Simulation Software (Proteus): Used for initial design and testing simulations.
* Microcontroller: Controls relay switching based on voltage levels.
* Altium Designer: Used for designing the power supply and stabilizer PCBs.
* Electronic Components: Transformers, relays, operational amplifiers, capacitors, resistors, diodes, and LCD panels.

## Conclusion
The Automatic Voltage Stabilizer project demonstrates the application of theoretical knowledge in electrical engineering to develop a practical solution for maintaining stable power supply to sensitive electronic equipment.
